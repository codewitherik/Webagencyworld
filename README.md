# Web Agency World

Web Agency World is a **professional multi-page web agency website** showcasing services, pricing, team, portfolio, contact, and community (Discord) integration.  

## Features

- **Multi-Page Structure**: Home, Services, Pricing, Contact, Imprint, Privacy  
- **Interactive Hero Section**: Typed.js, Three.js 3D canvas, GSAP animations  
- **Responsive Design**: Desktop & Mobile optimized  
- **Modern Animations**: Animate.css + GSAP ScrollTrigger  
- **Pricing & CTA**: Clear, responsive pricing cards with call-to-action  
- **Counters, Charts, Circular Progress Bars** for stats  
- **Contact Form**: Integrated with [Getform.io](https://getform.io)  
- **Leaflet Map**: Interactive map with custom markers  
- **Community Integration**: Discord join button & links  
- **SEO Optimized**: Meta tags, Open Graph, canonical URL  
- **Global CSS & Reset**: Modern typography and color palette  

## Tech Stack

- HTML5, CSS3, JavaScript  
- Three.js for 3D elements  
- GSAP for scroll animations  
- Animate.css for entrance effects  
- Typed.js for hero typewriter animations  
- Leaflet.js for interactive maps  
- Getform.io for backend-free contact forms  
- Font Awesome for icons
